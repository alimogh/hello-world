工具升级说明

*****重要通知: 服务器域名从 v2.ifreesky.cn 更新为 test.freesky.ltd

具体操作如下：

一、Windows / 苹果电脑 MAC 客户端

1.Windows系统关闭 v2ray.exe 黑窗口，苹果电脑Mac系统关闭v2ray

2.用记事本打开 config.json 文件， 找到 "v2.ifreesky.cn" 替换为 "test.freesky.ltd" ，保存。

***注意：双引号里面不要有空格！ 不要有空格！ 不要有空格！

3.重新启动v2ray (v2ray.exe)

二、手机 / 平板电脑Pad

打开APP，在 服务器 Server / 主机 Host / 地址 Address 输入框中 找到 v2.ifreesky.cn 替换为 test.freesky.ltd

***注意：test.freesky.ltd 前后不要有空格！ 不要有空格！ 不要有空格！
